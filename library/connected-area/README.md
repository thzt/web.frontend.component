存在一个正整数的坐标系，输入一个数组，数组的元素都是这个坐标系的坐标 [x, y]，现在需要将输入的坐标数组拆分为若干个子数组，具有相邻关系（上下左右相邻）的坐标需要分配到同一个子数组中。

input 内坐标是无序的
```
const input = [
  [0, 0],
  [1, 1],
  [3, 1],
  [1, 2],
  [3, 2],
  [2, 3],
  [1, 4],
  [2, 4],
  [4, 4],
  [4, 5],
  [3, 4]
]
```

output:
```
[
 [[0, 0]],
 [[1,1], [1,2]],
 [[3,1], [3,2]],
 [[1,4], [2,4], [2,3], [3,4], [4,4], [4,5]]
]
```
